隐私声明摘要

  • 我们不会主动获取、收集、上传你的代码文件、文档内容或个人信息。
  • 我们不会主动联网，不依赖网络完成导入、筛选、预览和导出。
  • 你主动选择的文件、目录或 ZIP 可能被复制到应用沙箱中做本地处理，但不会主动上传到开发者服务器。
  • 只有在你点击复制操作时，预览文本或诊断日志才会被写入系统剪贴板。
  • 当前版本不涉及个性化推荐、广告营销、第三方统计 SDK 或跨境传输。

1. 我们如何收集和使用您的个人信息

我们仅在有合法性基础的情形下处理信息。对于本应用而言，主要合法性基础是：为实现你主动发起的文件整理、预览和导出功能所必需。 当前版本不设置注册、登录、实名认证、会员体系或在线账户，因此通常不以建立用户档案的方式处理个人信息。

处理对象 来源 用途 处理方式

1.2 个性化推荐说明

当前版本不涉及个性化推荐。我们不会根据你的使用习惯、文件内容、设备信息或其他行为构建画像，也不会据此向你推荐内容、服务或功能。

1.3 广告和营销说明

当前版本不涉及广告投放、商业营销、短信通知、邮件营销、消息推送拉新或广告效果归因。我们不会为广告或营销目的处理你的个人信息。

2. 设备权限调用与系统能力说明

当前 Harmony 工程中，未看到网络、定位、通讯录、短信、相机、麦克风等与个人信息处理强相关的运行权限声明。 但为实现本地文件处理功能，应用会调用少量系统能力：

系统文件选择器

仅在你点击导入文件、导入目录、导入 ZIP 或选择导出位置时调用，用于获取你主动选择的路径。

本地文件读写

用于读取你选择的源码或 ZIP，并在本地生成预览内容和 DOCX 文档。

系统剪贴板

仅在你点击“复制预览”或“复制诊断日志”时写入内容，不会主动读取你的历史剪贴板。

系统备份/恢复声明

工程声明了备份扩展能力。是否发生备份、备份范围和去向取决于操作系统与用户设置，不代表我们通过自建服务器收集数据。

3. 对未成年人的保护

本应用是一款面向代码整理与文档导出的工具，不以未成年人为主要目标用户。若你是未成年人，建议在监护人指导下阅读本声明并使用本应用。 如果监护人认为未成年人向我们提供了不适当的信息，可通过本声明末尾的联系方式与我们联系。

4. 与第三方共享

当前版本原则上不与第三方共享你的个人信息、源码文件、ZIP 内容、DOCX 导出结果、预览文本或诊断日志，也不会主动向开发者服务器上传这些内容。

  • 不向广告平台共享。
  • 不向统计分析平台共享。
  • 不向社交平台或营销平台共享。
  • 不因正常使用流程向任何第三方云端传输你的源码内容。

5. 第三方 SDK

当前版本不接入用于广告、统计、用户画像、登录、推送或商业化变现的第三方 SDK。 Harmony 工程中使用的系统 Kit 属于操作系统或开发框架基础能力，不构成面向用户数据经营的第三方 SDK 集成场景。

6. 管理您的个人信息

由于当前版本不提供在线账号体系，你无需执行“注销账户”操作。你可以通过以下方式管理与本应用有关的数据：

  1. 不导入任何文件、目录或 ZIP。
  2. 自行删除已导出的 DOCX 文件。
  3. 清空应用内预览内容和诊断日志。
  4. 通过覆盖复制或系统操作清理剪贴板内容。
  5. 通过系统设置清除应用数据或卸载应用。

若你对与本应用有关的数据主体权利、疑问、意见或建议有进一步要求，可通过“如何联系我们”章节中的方式与我们联系。

7. 信息存储地点及期限

7.1 存储地点

当前版本不使用自建服务器保存你的个人信息。你主动选择的文件、临时副本、解压结果、预览内容及导出文档，主要保存在当前设备本地或应用沙箱中。

7.2 存储期限

运行期生成的预览内容和诊断日志通常仅在应用运行期间保留；导出的 DOCX 文件及你主动保存的内容，由你自行决定保留时长并自行删除。 沙箱中的临时副本将随应用数据清理、覆盖或卸载而失效或删除。

7.3 跨境处理声明

当前版本不涉及将你的个人信息传输并保存至中国境外服务器的处理活动。若未来版本引入云同步、账号体系或在线服务，我们会在功能上线前更新本声明。
